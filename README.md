# CrankApp Legal Pages

Static legal pages for CrankApp — hosted via GitHub Pages.

## Pages

| Page | URL |
|---|---|
| Home | `/` |
| Terms of Use | `/terms.html` |
| Privacy Policy | `/privacy.html` |

## Setup (GitHub Pages)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch** → `main` → `/ (root)`
4. Your pages will be live at `https://<your-username>.github.io/<repo-name>/`

## Customization

Before going live, update these placeholders:

- `support@crankapp.app` → your real support email
- Developer/company name (for legal contact section)
- Governing law jurisdiction if not Vietnam

## Languages

All pages support English (EN) and Vietnamese (VI) via a toggle button.
